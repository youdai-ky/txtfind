テキスト検索CLIツール

テキストファイル内の文字列を検索するシンプルなCLIツール．

## 概要（Overview）
txtfindは，指定した文字列をテキストファイルから検索し，一致した行を表示するコマンドラインツールである．

## 使い方（Usage）
txtfind pattern file

txtfind pattern file OPTIONS
